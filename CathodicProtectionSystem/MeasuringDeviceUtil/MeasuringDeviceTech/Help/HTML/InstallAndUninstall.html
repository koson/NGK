Установка программы

Поддерживаемые операционные системы:

  1. Windows 7
  2. Windows 7 Service Pack 1
  3. Windows Vista Service Pack 1
  4. Windows XP Service Pack 3

Поддерживаемые архитектуры:

x86

Аппаратные требования:

Рекомендуемый минимум:
процессор Pentium с тактовой частотой 1 ГГц или выше, 512 МБ оперативной памяти или больше

Предварительные требования:

  1. Установщик Windows 3.1 или более поздней версии

Пакет инсталляции содержит файлы:

  1. NGKDeviceTerminal.msi
  2. setup.exe
Для начала инсталляции ПО на компьютер запустите файл setup.exe. И следуйте инструкциям. Если на вашей системе не установлен Microsoft .NET Framework 4. То он будет загружен автоматически с сайта Microsoft. Если отсутствует интернет соединение, то перед установкой программы необходимо в ручную установить Microsoft .NET Framework 4, предварительно скачав его на www.microsoft.com. Так же необходимо предварительно установить установщик Windows Installer 3.1. Загрузить его можно по адресу www.microsoft.com

Удаление программы

Удаление программы. Открыть панель управления и запустить «Программы и компоненты» в Windows 7 или «Установка и удаление программ» в Windows XP. Выбрать NGK Devices Terminal и удалить.
