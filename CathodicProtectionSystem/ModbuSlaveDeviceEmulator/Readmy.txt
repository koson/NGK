Требование к системе:
2. Windows XP SP3 и выше.
1. Microsoft Framework 2.0

Установка ПО:
Не требует инсталляции. Для запуска программы достаточно скопировать в папку 
файлы архива и запустить .exe файл.

Настройка ПО:
1. Настройки приложения хранятся в файле ModbusNetworkEmulator.exe.config.
2. Настройки конфигурации сети Modbus находятся в файле config.xml (по умолчанию)
3. Если необходимо указать иной файл конфигурации сети, то для этого необходимо
	изменить параметр "PathToNetworkConfigFile" в файле ModbusNetworkEmulator.exe.config.
	Где необходимо указать полный путь к файлу конфигурации сети. Если данный параметр
	установлен в "", то используется настройка по умолчанию, т.е файл config.xml в папке приложения
4. Настроить COM-порт необходимо в файле ModbusNetworkEmulator.exe.config. Для этого 
	установить параметры "PortName", "PortName", "BaudRate", "Parity", "DataBits", "StopBits".
	Пример:
	<!--Наименование COM-порта, например COM1, COM2 и т.д.-->
    <add key="PortName" value="COM1"/>
    <!--Скорость обмена данными, например 9600, 19200 и т.д из стандартного ряда скоростей COM-порта-->
    <add key="BaudRate" value="19200"/>
    <!--Использование верификации фрейма. Значение Even, Odd, None-->
    <add key="Parity" value="Even"/>
    <!--Количество бит данных символа 7 или 8-->
    <add key="DataBits" value="8"/>
    <!--Количество стоп-бит фрейма One, Two, None-->
    <add key="StopBits" value="One"/>   
	
Работа с приложением
После запуска приложения и успешного создания сети из файла конфигурации сети. Можно отредактировать
конфигурацию сети. Для этого необходимо выбрать меню "Сеть"->"Конфигурация сети" при этом соединение
(COM-порт должен быть закрыт). Откроется окно с возможностью редактировать конфигурацию. Для добавления
изменения или удаляения элемента данных устройства выбирается соответствующая закладка и по клику правой
кнопки появиться контекстное меню, где можно выбрать соответствующее действие. Дабавить или удалить устройство
из сети можно в левом окне аналогичным способом. После того, как сеть отредактирована возвращаемся в основное
окно приложения. Для начала работы сети омечаем галочкой устройства, которые будут реагировать на запросы. Если
устройство не отмечно галочкой оно находиться в выключенном состоянии и на запросы реагировать не будет. Далее
необходимо открыть соединение через COM-порт. Для этого необходимо нажать кнопку "Старт" панели инструментов.
Соотвественно, остановить работу сети можно нажатием кнопки "Стоп".
Сохранить текущую конфигурацию можно следующими способами:
1. Выбрать меню "Файл"->"Сохранить". При этом текущая конфигурации сохраняется в файл из которого была
	создана сеть.
2. Выбрать меню "Файл"->"Сохранить как". При этом текущая конфигурация сохраняется в файл по указанному пути.
